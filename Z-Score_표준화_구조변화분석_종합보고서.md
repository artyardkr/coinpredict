## 1. 연구 배경 및 방법론

### 1.1 왜 Z-Score 표준화인가?

#### 기존 분석의 한계
```
문제: Chow Test의 F-통계량은 구조변화 "존재 여부"만 알려줌
한계: 변수 간 영향력 크기 비교 불가능

예시:
- 인플레이션 F = 5,801
- 금리(DFF) F = 5,778
→ 둘 다 구조변화 있음은 알지만, 어느 쪽이 더 중요한지 모름
```

#### Z-Score 표준화의 장점
```python
# 표준화된 계수(β)의 의미
β = 0.5 → X가 1 표준편차 ↑ 시, Y가 0.5 표준편차 ↑

모든 변수 동일 스케일(평균 0, 표준편차 1)
→ β의 절대값 = 영향력의 직접적 크기
→ 변수 간 영향력 순위 매기기 가능
```

### 1.2 방법론

#### Step 0: 데이터 표준화
```
1. 결측치 처리 (forward fill → backward fill → median)
2. Winsorization (상하위 1% 절사)
3. Z-score 변환

   z = (x - μ) / σ

검증:
✓ 평균: -0.000000 (≈ 0)
✓ 표준편차: 1.000292 (≈ 1)
```

#### Step 1-3: 3단계 검정
```
Step 1: Chow Test (ETF 승인일 기준 구조변화 검정)
Step 2: Quandt-Andrews Test (변화점 자동 탐지)
Step 3: CUSUM Test (변화 유형 분류: 급격 vs 점진)
```

#### Step 4: 표준화 계수 비교 분석 ⭐
```python
# 영향력 측정
impact_pre  = |β_pre|
impact_post = |β_post|
impact_change = impact_post - impact_pre

# 영향력 분류
VERY_HIGH  : |β| > 1.0
HIGH       : |β| > 0.5
MODERATE   : |β| > 0.3
LOW        : |β| > 0.1
VERY_LOW   : |β| ≤ 0.1

# 방향 변화
REVERSED   : 부호 전환 (+ → - 또는 - → +)
AMPLIFIED  : 같은 방향, 1.5배 이상 증폭
DAMPENED   : 같은 방향, 0.5배 이하 감소
STABLE     : 같은 방향, 비슷한 크기
```

## 7. 경제적 해석

### 7.1 비트코인의 자산 정체성 변화

#### ETF 이전 (2021-2023)
```
자산 특성: 독립적, 탈중앙화
주요 영향:
- 채굴 경제학 (Hash Rate, Difficulty)
- 온체인 지표 (거래량, 수수료)
- 투기적 수요

전통 금융과 관계: 약함 또는 무관
```

#### ETF 이후 (2024~)
```
자산 특성: 전통 금융시장 편입
주요 영향:
- 금리 정책 (SOFR, DFF) - 가장 강력
- 인플레이션 (CPIAUCSL)
- 거시경제 (GDP, 실업률)

전통 금융과 관계: 강하게 연동
역할: 인플레이션 헤지 + 위험자산
```

### 7.2 투자 시사점

#### 1. 포트폴리오 다각화 효과 감소
```
ETF 이전: BTC는 주식·채권과 낮은 상관관계 → 분산 효과 ↑
ETF 이후: 금리에 민감하게 반응 → 분산 효과 ↓

결론: 비트코인만으로 포트폴리오 분산 어려워짐
```

#### 2. 금리 사이클 의존성 증가
```
연준 금리 인상기: BTC 하락 압력 증가
연준 금리 인하기: BTC 상승 탄력 증가

→ 기존보다 금리 정책 추적 중요성 ↑
```

#### 3. 기관투자자 영향력 증대
```
ETF 자금 유입 → 전통 금융 로직 지배
개인 투자자 심리 < 기관투자자 매크로 전략

→ 온체인 지표보다 거시경제 지표 중요
```

## 8. 기술적 우수성

### 8.1 Z-Score 표준화의 가치 입증

| 측면 | 비표준화 분석 | Z-Score 표준화 분석 |
|------|-------------|-------------------|
| **구조변화 탐지** | ✓ 가능 | ✓ 가능 |
| **영향력 크기 비교** | ✗ 불가능 | ✓ **가능** |
| **변수 순위 매기기** | ✗ 불가능 | ✓ **가능** |
| **경제적 해석** | △ 제한적 | ✓ **명확함** |
| **방향 전환 탐지** | △ 수동 | ✓ **자동화** |

### 8.2 다중 검정 보정

```
Bonferroni 보정: 가장 보수적
- 129/137 변수 유의 (94.2%)
- Type I Error 엄격히 통제

FDR (Benjamini-Hochberg) 보정: 합리적 균형
- 133/137 변수 유의 (97.1%)
- False Discovery Rate 통제하면서 검정력 유지

→ 두 방법 모두 90% 이상 유의 = 결과 신뢰성 ↑
```

## 9. 결론

### 9.1 주요 발견 요약

1. **금융시장 편입의 증거**
   - 97.1%의 변수가 구조변화 경험
   - 금리 변수의 영향력 10배 이상 증폭

2. **자산 성격의 근본적 변화**
   - 38개 변수 방향 전환 (27.7%)
   - 인플레이션, GDP와 (+) 관계로 전환

3. **영향력 구조의 재편**
   - VERY_HIGH 영향력 변수 7.5배 증가
   - 채굴 경제학 → 거시경제 주도로 전환

4. **안정성 증가**
   - BTC 변동성 20.5% 감소
   - 하지만 금리 민감도는 극대화

### 9.2 향후 연구 방향

1. **변동성 전이(Spillover) 분석**
   - VAR 모형으로 방향성 있는 영향 측정
   - Granger 인과성으로 시간적 선후관계 파악

2. **동적 상관관계(DCC-GARCH)**
   - 시간에 따라 변하는 상관계수 추정
   - 레짐 스위칭 모형 검토

3. **고차원 네트워크 분석**
   - 137개 변수 간 연결망 구조 변화
   - 중심성(Centrality) 분석

impact_change > 0.5    # 큰 증가

impact_change < -0.5   # 큰 감소
