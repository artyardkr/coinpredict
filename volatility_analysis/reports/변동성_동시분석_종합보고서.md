## 📈 Chapter 1: GARCH 변동성 & Rolling Correlation

### 1.1 방법론

**GARCH(1,1) 모형:**
```
σ²ₜ = ω + α·ε²ₜ₋₁ + β·σ²ₜ₋₁

여기서:
- σ²ₜ: t시점 조건부 분산
- εₜ: 잔차 (수익률 충격)
- ω, α, β: 추정 파라미터
```

**분석 대상:** BTC, SPX, QQQ, VIX, GOLD, TLT, DXY (7개 핵심 자산)

## 📉 Chapter 2: VAR 분산분해 & Spillover Index

### 2.1 방법론

**VAR(1) 모형:**
```
Yₜ = c + Φ·Yₜ₋₁ + εₜ

여기서:
- Yₜ: 7개 자산 변동성 벡터
- Φ: 7×7 계수 행렬
- εₜ: 충격 벡터
```

**Diebold-Yilmaz Spillover Index:**
```
Spillover = (비대각선 요소 합 / 전체 분산 합) × 100

해석: 시스템 내에서 다른 자산으로 전이되는 변동성 비율
```

## 🔗 Chapter 3: Granger 인과관계

### 3.1 방법론

**Granger Causality Test:**
```
H₀: X는 Y를 Granger-cause 하지 않음
H₁: X는 Y를 Granger-cause 함

검정: F-test (VAR 모형 기반)
유의수준: α = 0.05
```

**해석:**
- X → Y 인과관계 = "X의 과거 값이 Y를 예측하는 데 도움"
- 통계적 선후행 관계 (경제적 인과관계 아님)

### 3.2 BTC 관련 인과관계

|------|---------|---------|---------|---------|------|
| 1 | **TLT** | 0.0049 ✅ | **VIX** | 0.0025 ✅ | VIX가 1순위로 |

### 3.3 획기적 발견: BTC → GOLD 인과관계 출현

```
ETF 이전: p = 0.1580 (비유의)
  → 금 가격이 BTC를 예측하지도, BTC가 금을 예측하지도 않음

ETF 이후: p = 0.0463 (유의!)
  → BTC가 금 가격을 2-5일 선행 예측!

의미:
✅ BTC가 안전자산 시장을 선도하기 시작
✅ "디지털 금" 담론이 실제 시장 구조로 구현됨
✅ BTC 급등/급락 → 2-5일 후 금 가격 움직임 예상 가능

투자 전략:
1. BTC +10% 급등 시
   → GLD (금 ETF) 매수 타이밍

2. BTC -10% 급락 시
   → 금 매도 또는 공매도 고려

3. BTC 변동성 확대 시
   → 금 시장 준비
```

## 🌊 Chapter 4: DCC-GARCH (동적 조건부 상관관계)

**DCC-GARCH:**
```
본 연구에서는 간소화된 DCC 적용:

1. 각 자산별 GARCH(1,1) 추정
```

## 🎲 Chapter 5: Markov Switching 레짐 분석

### 5.1 방법론

**Hamilton (1989) Markov Switching Model:**
```
yₜ = μ(sₜ) + εₜ
εₜ ~ N(0, σ²(sₜ))

여기서:
- sₜ: 레짐 상태 (0 or 1)
- μ(sₜ): 레짐별 평균 수익률
- σ²(sₜ): 레짐별 분산
- 레짐 전환 확률 자동 추정
```

### 5.2 레짐 특성

| 지표 | Regime 0 (저변동) | Regime 1 (고변동) |
|------|------------------|------------------|
| **평균 수익률 (일별)** | 0.001% | 0.235% |
| **연율화 수익률** | 0.17% | **59.27%** |
| **변동성 (일별)** | 1.34% | 4.19% |
| **연율화 변동성** | 21.25% | 66.53% |
| **샤프 비율** | 0.006 | 0.058 |
| **양의 수익 비율** | 48.6% | 52.1% |

**역설적 발견:**
```
❗ 변동성 높을수록 수익률도 높다!

Regime 1 (고변동):
- 수익률 38배 높음 (0.001% vs 0.235%)
- 샤프 비율 10배 높음 (0.006 vs 0.058)
- 양의 수익 확률도 높음 (48.6% vs 52.1%)

투자 함의:
✅ "No Pain, No Gain" 입증
✅ 변동성 = 기회
✅ 변동성 회피 전략 재고 필요
```

### 5.3 레짐 분포 변화

| 기간 | Regime 0 (저변동) | Regime 1 (고변동) |
|------|------------------|------------------|
| **전체 기간** | 1,027일 (59.9%) | 687일 (40.1%) |
| **ETF 이전** | 592일 (55.3%) | 478일 (44.7%) |
| **ETF 이후** | 435일 (67.5%) | 209일 (32.5%) |

**핵심 변화:**
```
저변동 레짐 비중: 55.3% → 67.5% (+12.2%p)

의미:
✅ 시장 안정화 뚜렷
✅ 극단적 변동성 감소
✅ 장기 투자 환경 개선

하지만:
⚠️ 고수익 기회도 감소
⚠️ Regime 1 비중 44.7% → 32.5% (-12.2%p)
```

### 5.4 레짐 전환 확률

#### **전체 기간**
| 전환 | 확률 | 의미 |
|------|------|------|
| Regime 0 → 0 | 74.0% | 저변동 지속 확률 |
| Regime 0 → 1 | 26.0% | 저변동 → 고변동 |
| Regime 1 → 0 | 26.0% | 고변동 → 저변동 |
| Regime 1 → 1 | 70.4% | 고변동 지속 확률 |

**평균 지속 기간:**
- Regime 0: 3.8일
- Regime 1: 3.4일

#### **ETF 이후 변화**
| 전환 | ETF 이전 | ETF 이후 | 변화 |
|------|---------|---------|------|
| Regime 0 → 0 | 72.5% | 67.3% | -5.2%p |
| Regime 1 → 1 | 74.1% | 67.4% | -6.7%p |

**평균 지속 기간 변화:**
- Regime 0: 3.6일 → 3.1일 (-14%)
- Regime 1: 3.9일 → 3.1일 (-21%)

**해석:**
```
✅ 레짐 전환이 더 빈번해짐
✅ 시장이 민첩하게 반응
✅ 변동성 클러스터링 감소

투자 시사점:
- 극단적 상황 지속 기간 짧아짐
- 손절매 더 빠르게
- 기회 포착도 빠르게
```

### 5.5 실전 투자 전략

#### **전략 1: 레짐 기반 동적 배분**

```python

if P(Regime 1) > 0.7:  # 고변동 레짐 확률 70% 이상
    # 고변동 = 고수익 기회
    btc_allocation = 40%
    risk_on = True

    action = "BTC 비중 확대"
    reason = "Regime 1은 연 59% 수익률, 변동성 수용"

elif P(Regime 0) > 0.7:  # 저변동 레짐 확률 70% 이상
    # 저변동 = 저수익
    btc_allocation = 20%
    risk_on = False

    action = "BTC 비중 축소, 안전자산 증가"
    reason = "Regime 0은 연 0.17% 수익률, 기회 적음"

else:
    # 불확실
    btc_allocation = 30%
    action = "중립 포지션"
```


#### **전략 3: ETF 후 안정성 활용**

```

   - 저변동 비중 증가 → 안심 보유
```
