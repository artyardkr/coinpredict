## 2. Kehinde et al. (2025)
**제목**: "Helformer: an attention-based deep learning model for cryptocurrency price forecasting"

**출처**: Journal of Big Data (2025) 12:81

### ✅ 우리 연구에서 활용한 부분

#### **(1) 시계열 데이터의 비정상성 문제 인식**
**논문의 기여**:
- Holt-Winters로 계절성/추세 제거
- Transformer로 장기 의존성 학습

**우리가 적용한 방법**:
```python
# 우리는 Transformer 대신 ElasticNet 사용
# 이유: 외삽(Extrapolation) 문제

문제 인식:
- Train 최댓값: $73,000
- Test 최댓값: $113,000
→ 트리 모델은 $73K 이상 예측 불가

해결책:
- ElasticNet (선형 모델)
- 본 적 없는 가격도 선형 외삽으로 예측 가능
```

**직접 영향받은 파일**:
- `step24_all_regression_models.py` - 10개 모델 비교
- `step25_v2_next_day_prediction.py` - ElasticNet 최종 선정

**PPT 슬라이드 16에서 언급**:
```
왜 ElasticNet이 최고? 선형 vs 트리

문제: 가격 외삽 (EXTRAPOLATION)
TRAIN: 최댓값 $73,000
TEST: 최댓값 $113,000

→ ElasticNet: 선형 함수로 범위 밖 예측 가능
→ Random Forest: 본 것만 예측 가능
```

#### **(2) Transfer Learning 개념 참고**
**논문의 기여**:
- 15개 코인에 동일 모델 적용 (Transfer Learning)
- 모델의 일반화 능력 검증

```python
- 2024 데이터에 적용 → 성능 하락

V2 모델 (ETF 후 재학습):
- 2022-2024 데이터로 학습
- 2024 데이터에 적용 → 성능 향상

→ 시장 구조 변화로 재학습 필요성 입증
```

**제목**: "Bitcoin price direction prediction using on-chain data and feature selection"

**출처**: Machine Learning with Applications 20 (2025) 100674

### ✅ 우리 연구에서 활용한 부분

#### **(1) 온체인 데이터 5개 카테고리**
**논문의 분류**:
1. Network Activity (네트워크 활동)

```python
# step6_onchain_data.py - 카테고리별 수집

4. Derivatives:
   - binance_open_interest (미결제약정)
```

#### **(2) Boruta Feature Selection**
**논문의 방법**:
- Boruta 알고리즘으로 중요 변수 선택
- Random Forest 기반 중요도 측정

**우리가 적용한 방법**:
```python
# 우리는 ElasticNet의 L1 정규화 사용
# 이유: 선형 모델 + 자동 변수 선택

ElasticNet = Lasso (L1) + Ridge (L2)

L1 정규화:
- 중요하지 않은 변수 계수 → 0
- 자동으로 변수 선택 (Feature Selection)

L2 정규화:
- 다중공선성 해결
- 비슷한 변수들 (S&P vs Nasdaq) 균형
```

**관련 파일**:
- `ElasticNet_변수분석.md`
- `Step25_ElasticNet_변수분석.md`
- `elasticnet_xai_analysis_v2.py` - SHAP으로 변수 중요도 분석

**PPT 슬라이드 16에서 설명**:
```
ElasticNet 특징:

특징 1: '탈락' (Lasso)
→ 영향력 없는 데이터 제거 (변수 선택)

특징 2: '조절' (Ridge)
→ 비슷한 변수 영향력 조절 (다중공선성 해결)

← Boruta 대신 ElasticNet으로 Feature Selection
```

```python
- 실제 가격 예측

→ 가격 예측이 실전에서 더 유용
```
